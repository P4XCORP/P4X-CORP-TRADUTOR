# P4X CORP TRADUDOR v2.0

## Características

- **Tradução Automática**: Copie qualquer texto (Ctrl+C) e veja a tradução instantaneamente
- **Multi-idioma**: Suporta 18+ idiomas incluindo Português, Inglês, Espanhol, Russo, Árabe, Hindi, Japonês, Chinês e mais
- **Modo Escrever**: Componha mensagens em um idioma e traduza para outro antes de enviar
- **Configurável**: Escolha idiomas de origem e destino para receber e enviar mensagens
- **Always on Top**: Overlay flutuante que fica sobre todas as janelas

## Como Usar

1. Execute o aplicativo
2. Copie qualquer texto (Ctrl+C) - a tradução aparece automaticamente
3. Use o botão "Escrever" para compor mensagens e traduzi-las
4. Configure os idiomas no botão "Config"

## Atalhos

- `Ctrl+Shift+T` - Mostrar/Ocultar janela
- `Ctrl+Shift+C` - Forçar tradução do clipboard
- `Ctrl+C` - Tradução automática (copie qualquer texto)

## Idiomas Suportados

Português, Inglês, Espanhol, Francês, Alemão, Italiano, Russo, Árabe, Hindi, Japonês, Coreano, Chinês, Turco, Holandês, Polonês, Ucraniano, Tailandês, Vietnamita
